Assignment name  : max
Expected files   : max.c
Allowed functions:
--------------------------------------------------------------------------------

Écrivez la fonction suivante :

int		max(int* tab, unsigned int len);

Le premier paramètre est un tableau d'entiers (int), le second est le nombre 
d'éléments dans le tableau.

La fonction retourne le plus grand nombre trouvé dans le tableau.

Si le tableau est vide, la fonction retourne 0.
